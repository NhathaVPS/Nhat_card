Khái niệm bộ trọn trong JS 

        trong css
        tagName : p{}
        id : #id {}
        class css : .className 

        trong JS 
        chọn viết theo ID là : document.getElementByid("id")
        chú ý không truyền dấu thăng (#)

        muốn thay đổi nội dung html bên trong 1 thẻ theo id
        document.getElementByid("id").innerHTML = "giá trị mới";
        thay đổi văn bản bên trong 
        document.getElementByid("id").innerText = "giá trị mới";
